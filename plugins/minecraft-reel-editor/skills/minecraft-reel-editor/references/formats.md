# File formats

## Transcript

`transcript.json` is generated by `transcribe.py`.

```json
{
  "schema_version": 1,
  "source": "/absolute/path/video.mp4",
  "language": "en",
  "duration": 300.0,
  "backend": "mlx-whisper",
  "model": "mlx-community/whisper-large-v3-turbo",
  "segments": [],
  "words": [
    {
      "id": "w000001",
      "segment_id": "s0001",
      "start": 1.2,
      "end": 1.52,
      "text": "Today"
    }
  ]
}
```

Word IDs are the immutable edit anchors.

## Selection

Create `selection.json` manually after analyzing the transcript.

```json
{
  "schema_version": 1,
  "target_duration_seconds": 30,
  "clips": [
    {
      "id": "hook",
      "role": "hook",
      "start_word": "w000120",
      "end_word": "w000131",
      "layout": "center_crop",
      "pause_mode": "compress",
      "rationale": "Strongest complete statement of the result."
    },
    {
      "id": "payoff",
      "role": "payoff",
      "start_word": "w000245",
      "end_word": "w000259",
      "layout": "full_frame",
      "pause_mode": "preserve",
      "tail_padding": 0.8,
      "rationale": "The redstone result occurs after the final word."
    }
  ],
  "omissions": [
    "Discarded two false starts before the hook.",
    "Removed an unrelated tangent about resource gathering."
  ]
}
```

Allowed `role` values are `hook`, `setup`, `bridge`, `escalation`, and `payoff`.
Allowed `layout` values are `center_crop` and `full_frame`.
Allowed `pause_mode` values are `compress` and `preserve`.
`head_padding` defaults to 0.08 seconds and may not exceed 1 second.
`tail_padding` defaults to 0.08 seconds and may not exceed 3 seconds.

## Edit plan

`prepare_proposal.py` generates `edit-plan.json`. Do not hand-author derived text or timestamps. The renderer requires:

```json
{
  "schema_version": 1,
  "approved": true,
  "approval": {
    "approved_by": "user",
    "approved_at": "ISO-8601 timestamp"
  },
  "clips": []
}
```

`mark_approved.py` sets the approval fields only after the user explicitly accepts the displayed proposal.
